# Trackify 📝

Trackify is a simple Django-based task management web application that allows users to create, edit, complete, and manage daily tasks.

---

## 🚀 Features

- Add new tasks  
- Edit existing tasks  
- Mark tasks as completed  
- Delete tasks  
- Beginner-friendly clean UI  
- SQLite database integration  

---

## 🛠 Tech Stack

- Python  
- Django  
- HTML  
- CSS  
- SQLite  

---

## ⚙️ Installation & Setup

1. Clone the repository  
   git clone https://github.com/burgoju-lahari/Trackify.git  

2. Navigate into the project folder  
   cd Trackify  

3. Create a virtual environment  
   python -m venv venv  

4. Activate the virtual environment  

   On Windows:  
   venv\Scripts\activate  

   On Mac/Linux:  
   source venv/bin/activate  

5. Install dependencies  
   pip install django  

6. Run migrations  
   python manage.py migrate  

7. Start the server  
   python manage.py runserver
