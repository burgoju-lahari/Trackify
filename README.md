# 🌻 Trackify

Trackify is a clean and beginner-friendly task management web application built using Django.
It allows users to manage daily tasks with a simple, intuitive interface.

---

## 🌐 Live Demo

👉 https://trackify-ntal.onrender.com

## 🚀 Features

* User Signup & Login 🔐
* Add new tasks 📝
* Edit existing tasks ✏️
* Mark tasks as completed ✅
* Delete tasks 🗑️
* User-specific task management
* Clean and responsive UI

## ⚙️ Installation & Setup

1. Clone the repository
   git clone https://github.com/burgoju-lahari/Trackify.git

2. Navigate into the project folder
   cd Trackify

3. Create a virtual environment
   python -m venv venv

4. Activate the virtual environment

   **Windows:**
   venv\Scripts\activate

   **Mac/Linux:**
   source venv/bin/activate

5. Install dependencies
   pip install -r requirements.txt

6. Run migrations
   python manage.py migrate

7. Start the server
   python manage.py runserver
